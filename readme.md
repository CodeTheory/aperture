## Requirements

- macOS 10.12+
- Xcode 12+
- Swift 5.3+

## Install

#### SwiftPM

```swift
.package(url: "https://github.com/wulkano/Aperture", from: "0.3.0")
```

#### Carthage

```
github "wulkano/Aperture"
```

#### CocoaPods

```ruby
pod 'Aperture'
```
